knuth_taocp_canonical

Edition-specific, executable transcriptions of Knuth’s TAOCP algorithms.

This repository provides canonical MIXAL implementations of selected algorithms from The Art of Computer Programming, preserving:

Original line numbers and labels
Edition-specific provenance (1st, 2nd editions, etc.)
Verified execution under the GNU MDK MIX emulator
Highlights
Faithful transcription of Knuth’s algorithms
Canonical reference implementations for research, teaching, or experimentation
Edition-specific filenames 
Repository Layout
vol1_1ed/     # Volume 1, First Edition
vol1_2ed/     # Volume 1, Second Edition
vol2_1ed/     # Volume 2, First Edition
vol3_2ed/     # Volume 3, Second Edition
...
Files follow the naming convention:
knuth_taocp_v\<volume\>_\<edition\>_p\<page\>_\<algorithm\>.mixal

Example:

knuth_taocp_v3_2ed_p117_algorithm_q_quicksort.mixal
